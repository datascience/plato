To compile and use from the command line: 

1. Make sure the CLASSPATH includes this folder, lib\bootstrapconnector.jar, 
   and juh.jar, jurt.jar, jut.jar, officebean.jar, ridl.jar, unoil.jar from 
   Open Office.  

2. javac pdfcm\Main.java pdfcm\PDFConvert.java

3. java pdfcm.Main [-m mergeFile] [-d] file1 [file2 [file3 ...]]
